# 东软颐养中心管理系统 — 全功能演示脚本

> 录制用 | 全模块覆盖 | 15 项创新 | ~6 分钟 | 王建国贯穿

---

## 第一幕：仪表盘与老人管理 · Admin（~45s）

1. 输入 `admin` / `admin123` 登录 → 仪表盘 5 张统计卡片
2. 鼠标划过卡片 → 悬停变色，**讲解**："卡片即导航入口"
3. 点击 **「在住老人」** → 一键跳转老人管理
4. 搜索框输入 `张` → 姓名/身份证/电话三合一模糊搜索
5. 护理等级 `一级护理` + 状态 `在住` → 组合筛选
6. 双击任意行 → 弹窗 17 项完整信息含床位号 → 关闭
7. 点 **「编辑信息」** → 可修改全部字段、更换床位
8. 点 **「设置护理等级」** → 选等级级联添加服务项目

> **创新点**：仪表盘一键跳转、多维度组合筛选、全局双击查看详情

---

## 第二幕：入住登记 · Nurse 端（~35s）

9. 退出 Admin → 登录 nurse 账号
10. 左侧点 **「老人信息」** → 点击 **「老人入住」**
11. 姓名 `王建国`，身份证 `120101195006151232`
12. 自动提取出生日期/年龄/性别 → 三字段锁定不可编辑
13. 电话 `13800001234`，选床位 → 点 **「入住」**
14. 回到列表双击王建国 → 验证信息，关闭

> **创新点**：身份证 GB11643 校验、自动提取+锁定、床位号自动补齐

---

## 第三幕：床位管理 · Admin 端（~30s）

15. 切回 Admin，左侧点 **「床位管理」**
16. 顶部统计栏：总床位 / 空闲 / 占用 / 外出
17. 楼层筛选 `3楼` → 三色编码：绿=空闲、红=入住、橙=外出
18. 找有空床房间，点 **「添加床位」** → 选房间 → 输入 `4` → 自动补齐 `301-4`
19. FlowPane 流式布局自适应排列

> **创新点**：床位三色看板、楼层筛选、床位号自动补齐、FlowPane 流式布局

---

## 第四幕：护理等级与项目 · Admin 端（~30s）

20. 左侧点 **「护理等级」** → 5 个等级：自理型/一级/二级/三级/医疗专护
21. 点 `一级护理` → 查看已配置的服务项目列表
22. 点 **「配置护理项目」** → 可增删关联项目
23. 左侧点 **「护理项目」** → 全部项目列表，按类别自动编号（LZ/YL/KF）
24. 点 **「新增护理项目」** → 选类别 → 自动生成编号 → 填写价格 → 保存

> **创新点**：等级-项目级联配置、自动编号

---

## 第五幕：服务关注 · Admin 端（~30s）

25. 左侧点 **「服务关注」** → 搜索选王建国
26. 下方黄色提示条显示：口味、过敏原、忌口
27. 表格状态列动态计算：正常/到期/耗竭
28. 点 **「购买护理项目」** → 选项目 → 数量填 3 → 到期日期 → 购买
29. 切换到其他老人 → 饮食偏好提示条实时切换
30. 双击购买记录行 → 查看老人详情

> **创新点**：饮食偏好联动提示条、状态动态计算

---

## 第六幕：护理记录联动 · Nurse 端（~35s）

31. 切 Nurse，左侧点 **「护理记录」** → 列表显示：老人姓名、项目、时间、数量
32. 点 **「新增护理记录」** → 选王建国
33. 项目下拉只显示已购买且剩余>0的项目，标注"(剩余3次)"
34. 选项目 → 绿色提示"剩余次数：3，本次最多可执行3次"
35. 数量 1 → 提交 → 列表新增
36. 再点新增 → 同一项目剩余已减为 2 → 体现扣减
37. 次数用完后该项目不再出现在下拉列表中

> **创新点**：护理记录-服务关注联动扣减、剩余次数实时提示

---

## 第七幕：膳食偏好智能推荐 · Nurse 端（~35s）

38. 左侧点 **「膳食偏好」** → 上方表格显示 11 位老人膳食偏好
39. 点击陈大明 → 黄色提示条"过敏：鸡蛋 | 忌口：忌油腻、忌辛辣、忌生冷"
40. 下方食物表格自动过滤 → 标题"已为您过滤，共 N 款适合"
41. 番茄炒蛋、鸡蛋羹被排除（鸡蛋过敏映射"蛋"）→ 含辣含油炸被排除
42. 切换王建国 → 过敏：花生 → 坚果类被排除；忌口：少糖 → 甜食被排除
43. 38 款食物 6 大类别（荤菜/素菜/主食/汤品/水果/零食），过滤后每人专属推荐

> **创新点**：忌口+过敏原智能过滤、18组关键词映射、过敏原扩展匹配

---

## 第八幕：膳食管理双向关联 · Admin 端（~25s）

44. 切 Admin，左侧点 **「膳食管理」** → 上方老人选择器选王建国
45. 黄色提示条显示王建国膳食偏好（口味/过敏原/忌口/建议）
46. 食物表格含名称/类别/营养/单价列 → 支持增删改
47. 选陈大明 → 提示条切换为陈大明偏好 → 过敏：鸡蛋

> **创新点**：膳食管理双向关联、Admin-Nurse 膳食数据互通

---

## 第九幕：管家分配 · Admin 端（~20s）

48. 左侧点 **「管家分配」** → 下拉选管家 → 展示其服务客户
49. 点 **「添加服务客户」** → 列表显示未分配的在住老人
50. 选择王建国 → 分配完成，服务类型 `日常护理`

> **创新点**：一键分配、自动过滤已分配老人

---

## 第十幕：员工管理 · Admin 端（~20s）

51. 左侧点 **「员工管理」** → 列表展示所有员工
52. 姓名/职位/电话/入职日期/状态
53. 点 **「新增员工」** → 填写信息 → 提交
54. 选中员工 → **「编辑」** 或 **「删除」**

---

## 第十一幕：外出申请与审批 · 双端（~40s）

55. 切 Nurse，左侧点 **「外出申请」** → **「新增外出申请」**
56. 选王建国 → 选明天日期 → 预计归日自动 +1 天
57. 事由 `家属探访` → 提交 → `待审批`
58. 切 Admin，左侧点 **「外出手续」** → 看到王建国申请，双击确认信息
59. 点 **「审批通过」** → 二次确认弹窗 → 点「是」→ 状态变 `通过`
60. 再次点审批 → 提示"已审批，无法再次审批"
61. 切 Nurse，选王建国 → **「登记归来」** → 实际归时自动填入，床位恢复

> **创新点**：归日自动+1、审批二次确认+锁定

---

## 第十二幕：健康记录 · Admin 端（~20s）

62. 左侧点 **「健康记录」** → 下拉选王建国 → 筛选记录
63. 血压/心率/血糖/体重/体温/备注 7 列
64. 点 **「登记健康记录」** → 为王建国录入血压 130/80 等 → 列表新增
65. 双击行 → 查看王建国详情

> **创新点**：双端共享健康数据、双击详情

---

## 第十三幕：用户管理 · Admin 端（~15s）

66. 左侧点 **「用户管理」** → 展示所有用户
67. 账号/姓名/角色/密保问题
68. 仅管理员端可见此模块

---

## 第十四幕：退住申请与审批 · 双端压轴（~35s）

69. 切 Nurse，左侧点 **「退住申请」** → **「新增退住申请」**
70. 选王建国 → `正常退住` → 填日期 → 提交 → `待审批`
71. 切 Admin，左侧点 **「退住管理」** → 王建国 `待审批`
72. 双击确认信息 → **「审批通过」** → 二次确认
73. 王建国状态变 `退住`，床位自动释放为 `空闲`
74. 点 **「仪表盘」** → 在住老人 -1，床位使用 -1

> **创新点**：退住-床位联动、审批二次确认

---

## 第十五幕：操作日志与撤销 · Admin 端收尾（~30s）

75. 左侧点 **「操作日志」** → 8 列：操作人、角色、类型、目标、详情、时间、状态、跳转
76. 按时间倒序，刚才全流程操作均可追溯
77. 找到一条编辑操作 → 点 **「撤销」** → 弹出确认 → 数据回滚 → 标记"已撤销"
78. 已撤销操作不可重复撤销
79. 找到任一条 → 点 **「跳转」** → 一键导航到对应功能模块

> **创新点**：ThreadLocal 无侵入审计、JSON 快照撤销、一键导航
