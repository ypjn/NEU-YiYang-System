# 东软颐养中心管理系统 — 全功能演示脚本

> 录制用 | 10幕 | ~5分钟 | Admin↔Nurse 交替 | 王建国贯穿

---

## 第一幕：仪表盘与老人管理 · Admin（~50s）

1. 输入 `admin` / `admin123` 登录 → 仪表盘 5 张统计卡片
2. 鼠标划过卡片 → 悬停变色，"卡片即导航入口"
3. 点击 **「在住老人」** → 跳转老人管理
4. 搜索框输入 `张` → 姓名/身份证/电话三合一模糊搜索
5. 护理等级 `一级护理` + 状态 `在住` → 组合筛选
6. 双击任意行 → 弹窗 17 项信息含床位号 → 关闭
7. 点 **「编辑信息」** → 可修改全部字段、更换床位

> **创新点**：仪表盘一键跳转、多维度组合筛选、全局双击查看详情、老人姓名列

---

## 第二幕：入住登记 · Nurse 端（~40s）

8. 退出 → 登录 nurse 账号
9. 左侧点 **「老人信息」** → 点击 **「老人入住」**
10. 姓名 `王建国`，身份证 `120101195006151232`
11. 自动提取出生日期/年龄/性别 → 三字段锁定不可编辑
12. 电话 `13800001234`，选床位 → 点 **「入住」**
13. 回到列表双击王建国 → 验证信息，关闭

> **创新点**：身份证 GB11643 校验、自动提取+锁定、床位号自动补齐

---

## 第三幕：床位管理 · Admin 端（~30s）

14. 切回 Admin，左侧点 **「床位管理」**
15. 顶部统计栏：总床位 / 空闲 / 占用 / 外出
16. 楼层筛选 `3楼` → 三色编码：绿=空闲、红=入住、橙=外出
17. 找有空床房间，点 **「添加床位」** → 选房间 → 输入 `4` → 自动补齐 `301-4`

> **创新点**：床位三色看板、楼层筛选、自动补齐、FlowPane 流式布局

---

## 第四幕：护理等级与项目 · Admin 端（~25s）

18. 左侧点 **「护理等级」** → 展示 5 个等级
19. 点 `一级护理` → 查看已配置的服务项目
20. 点 **「配置护理项目」** → 可增删关联项目
21. 左侧点 **「护理项目」** → 全部项目及价格，按类别自动编号

> **创新点**：等级-项目级联配置、自动编号

---

## 第五幕：服务关注 + 护理联动 · 双端（~50s）

22. Admin 左侧点 **「服务关注」** → 搜索选王建国
23. 黄色提示条显示：口味/过敏原/忌口
24. 点 **「购买护理项目」** → 选项目 → 数量 3 → 购买
25. 切 Nurse，左侧点 **「护理记录」** → 点 **「新增护理记录」**
26. 选王建国 → 项目下拉只显示已购且剩余>0的项目，标注"剩余3次"
27. 数量 1 → 提交 → 列表新增
28. 再点新增 → 同一项目剩余已减为 2 → 体现联动扣减
29. 双击任意记录 → 查看王建国详情

> **创新点**：服务联动扣减、饮食偏好提示条、剩余次数提示、老人姓名列

---

## 第六幕：膳食偏好智能推荐 · Nurse 端（~30s）

30. 左侧点 **「膳食偏好」** → 上方 11 位老人偏好表格
31. 点击陈大明 → 提示条"过敏：鸡蛋 | 忌口：忌油腻、忌辛辣、忌生冷"
32. 下方食物表格自动过滤 → 标题"已为您过滤，共 N 款适合"
33. 番茄炒蛋、鸡蛋羹被排除（鸡蛋过敏）→ 含辣含油炸食物被排除
34. 切换王建国 → 过敏：花生 → 坚果类食物被排除
35. 38 款食物 6 大类别，每人看到专属推荐

> **创新点**：忌口+过敏原智能过滤、18组关键词映射、过敏原扩展匹配

---

## 第七幕：外出申请与审批 · 双端（~40s）

36. 切 Nurse，左侧点 **「外出申请」** → **「新增外出申请」**
37. 选王建国 → 选明天日期 → 预计归日自动 +1 天
38. 事由 `家属探访` → 提交 → `待审批`
39. 切 Admin，左侧点 **「外出手续」** → 看到王建国申请
40. 双击行确认信息 → 点 **「审批通过」** → 二次确认 → 状态变 `通过`
41. 再次点审批 → 提示"已审批，无法再次审批"
42. 切 Nurse，选王建国 → **「登记归来」** → 实际归时自动填入

> **创新点**：归日自动+1、审批二次确认+锁定、归来一键登记

---

## 第八幕：健康记录与管家分配 · Admin 端（~25s）

43. 左侧点 **「健康记录」** → 下拉选王建国 → 血压/心率/血糖/体重/体温
44. 点 **「登记健康记录」** → 为王建国录入 → 列表新增
45. 双击行查看王建国详情
46. 左侧点 **「管家分配」** → 下拉选管家 → 展示服务客户
47. 点 **「添加服务客户」** → 选王建国 → 分配完成

> **创新点**：老人姓名列、双击详情、一键分配

---

## 第九幕：退住申请与审批 · 双端压轴（~35s）

48. 切 Nurse，左侧点 **「退住申请」** → **「新增退住申请」**
49. 选王建国 → `正常退住` → 填日期 → 提交 → `待审批`
50. 切 Admin，左侧点 **「退住管理」** → 王建国 `待审批`
51. 双击确认信息 → **「审批通过」** → 二次确认
52. 王建国状态变 `退住`，床位释放为 `空闲`
53. 点 **「仪表盘」** → 在住老人 -1，床位使用 -1

> **创新点**：数据联动、审批二次确认、老人姓名列

---

## 第十幕：操作日志与收尾 · Admin 端（~25s）

54. 左侧点 **「操作日志」** → 8 列展示，按时间倒序
55. 刚才所有操作均可追溯：操作人、角色、操作类型、目标、详情
56. 找到一条编辑操作 → 点 **「撤销」** → 数据回滚
57. 找到任一条 → 点 **「跳转」** → 一键导航到对应模块
58. 已撤销操作不可重复撤销

> **创新点**：ThreadLocal 无侵入审计、JSON 快照撤销、一键导航、全实体覆盖
