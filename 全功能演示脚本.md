# 东软颐养中心管理系统 — 全功能演示脚本

> 组长录制用 | 覆盖全模块+全创新点 | 双端交替

---

## 第一幕：系统总览 · Admin 仪表盘（~30s）

1. 输入 `admin` / `admin123` 登录
2. 仪表盘 5 张统计卡片：在住老人 / 床位使用 / 外出登记 / 员工人数 / 护理记录
3. 鼠标划过卡片 → 悬停变色
4. **讲解**：卡片即导航入口，点任意卡片直达对应模块
5. 点击 **「在住老人」** → 跳转老人管理

> **创新点**：仪表盘一键跳转

---

## 第二幕：老人管理（~40s）

6. 搜索框输入 `张` → 姓名/身份证/电话三合一模糊搜索
7. 护理等级下拉选 `一级护理` → 组合筛选
8. 状态下拉选 `在住`
9. 双击任意老人行 → 弹窗显示 17 项完整信息 + 床位号
10. 关闭弹窗，点 **「编辑信息」** → 可修改全部字段、更换床位
11. 点 **「设置护理等级」** → 选等级会级联添加对应服务项目

> **创新点**：多维度组合筛选、全局双击查看详情

---

## 第三幕：入住登记 · Nurse 端（~40s）

12. 退出 → 登录 nurse 账号
13. 左侧点 **「老人信息」** → 点击 **「老人入住」**
14. 姓名 `王建国`
15. 身份证 `120101195006151234` → 自动提取出生日期/年龄/性别，三字段锁定
16. 电话 `13800001234`，选床位
17. 点 **「入住」** → 完成
18. 回到列表双击王建国，验证信息，关闭

> **创新点**：身份证前置 + 自动提取 + 锁定、床位号自动补齐

---

## 第四幕：床位管理 · Admin 端（~30s）

19. 切回 Admin，左侧点 **「床位管理」**
20. 顶部统计栏：总床位 / 空闲 / 占用 / 外出
21. 楼层筛选下拉选 `3楼` → 仅看三楼
22. **讲解**：三色编码 — 绿=空闲、红=入住、橙=外出
23. 找有空床的房间，点 **「添加床位」** → 选房间 → 输入数字 `4`
24. 自动补齐为 `301-4` → 确认添加

> **创新点**：色块可视化看板、楼层筛选、床位号自动补齐、FlowPane流式布局

---

## 第五幕：护理等级与项目 · Admin 端（~30s）

25. 左侧点 **「护理等级」**
26. 展示 5 个等级：自理型 / 一级 / 二级 / 三级 / 医疗专护
27. 点 **「一级护理」** → 查看下方服务项目列表
28. 左侧点 **「护理项目」** → 展示全部服务项目及价格

---

## 第六幕：护理记录 · Nurse 端（~30s）

29. 切回 Nurse，左侧点 **「护理记录」**
30. 列表显示：老人姓名（非ID）、项目代码、执行时间、数量、护工
31. 双击任意行 → 查看老人详情
32. 点 **「新增护理记录」** → 选老人 → 选项目 → 填数量 → 提交

> **创新点**：老人姓名列替换老人ID、双击查看详情

---

## 第七幕：服务关注 · Admin 端（~30s）

33. 切回 Admin，左侧点 **「服务关注」**
34. 搜索/选择老人 → 表格显示购买记录，第一列为老人姓名
35. 下方黄色提示条 → 显示该老人的口味、过敏原、忌口
36. 双击购买记录行 → 查看老人详情
37. 点 **「购买护理项目」** → 选项目 → 填数量 → 购买
38. 状态列动态计算：正常/到期/耗竭

> **创新点**：老人姓名列、饮食偏好提示条、双击查看详情、状态动态计算

---

## 第八幕：管家分配 · Admin 端（~20s）

39. 左侧点 **「管家分配」**
40. 下拉选管家 → 展示其服务客户
41. 第一列显示老人姓名
42. 点 **「添加服务客户」** → 从未分配老人中选择
43. 双击行查看老人详情

> **创新点**：老人姓名列、一键分配

---

## 第九幕：员工管理 · Admin 端（~20s）

44. 左侧点 **「员工管理」**
45. 列表展示所有员工（姓名/职位/电话/入职日期/状态）
46. 点 **「新增员工」** → 填写信息提交
47. 选中员工 → **「编辑」** 或 **「删除」**

---

## 第十幕：外出申请 · Nurse 端（~30s）

48. 左侧点 **「外出申请」**
49. 点 **「新增外出申请」** → 选王建国
50. 选明天日期 → **预计归日自动 +1 天**
51. 事由 `家属探访` → 提交 → 状态 `待审批`

> **创新点**：外出日期变化归日自动+1

---

## 第十一幕：外出审批 · Admin 端（~30s）

52. 切回 Admin，左侧点 **「外出手续」**
53. 列表第一列显示老人姓名（非ID）
54. 看到王建国申请，双击行确认信息
55. 点 **「审批通过」** → 弹出二次确认
56. 点 **「是」** → 审批完成，床位变 `外出`
57. 再次点审批 → 提示 `已审批，无法再次审批`

> **创新点**：老人姓名列、审批二次确认、审批后锁定

---

## 第十二幕：归来 + 退住 · Nurse 端（~30s）

58. 切回 Nurse，左侧点 **「外出申请」**
59. 选王建国 → **「登记归来」** → 实际归时自动填入，床位恢复
60. 左侧点 **「退住申请」** → **「新增退住申请」**
61. 选王建国 → 正常退住 → 提交 → 待审批

---

## 第十三幕：退住审批 · Admin 端（~20s）

62. 切回 Admin，左侧点 **「退住管理」**
63. 列表显示老人姓名，看到王建国待审批
64. 点 **「审批通过」** → 确认 → 老人变 `退住`，床位释放为 `空闲`

> **创新点**：老人姓名列、审批二次确认

---

## 第十四幕：健康记录 · 双端（~20s）

65. Admin 端点 **「健康记录」** → 老人姓名列 + 血压/心率/血糖/体重/体温
66. 下拉选老人 → 筛选该老人记录
67. 双击行 → 查看老人详情
68. 切 Nurse 端同样展示

> **创新点**：老人姓名列、双击详情

---

## 第十五幕：用户管理 · Admin 端（~15s）

69. 左侧点 **「用户管理」**
70. 展示所有用户（账号/姓名/角色/密保问题）
71. 管理员端才可见此模块

---

## 第十六幕：操作日志 · Admin 端收尾（~30s）

72. 左侧点 **「操作日志」**
73. 8列展示：操作人、角色、操作类型、目标、详情、时间、状态、跳转
74. 按时间倒序，刚才所有操作都可追溯
75. 找到一条编辑操作 → 点 **「撤销」** → 数据回滚
76. 找到任一条 → 点 **「跳转」** → 一键导航到对应功能模块
77. 已撤销操作不可重复撤销

> **创新点**：ThreadLocal 无侵入审计、JSON 快照撤销、一键跳转导航、全实体覆盖
