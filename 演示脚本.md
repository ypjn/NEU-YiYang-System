# 东软颐养中心管理系统 — 功能演示脚本

> 总时长约 3 分钟 | 双端切换（Admin ↔ Nurse）

---

## 第一幕：仪表盘与老人管理 · Admin 端（~50s）

**操作步骤**

1. 输入账号 `admin` / `admin123`，登录 → 进入仪表盘
2. 鼠标依次划过 5 张统计卡片 → **悬停变色 + 手型光标**
3. 讲解："每张卡片不仅是数据，更是功能入口"
4. 点击 **「在住老人」** 卡片 → 一键跳转老人管理
5. 搜索框输入 `张` → 模糊匹配姓名/身份证/电话
6. 护理等级下拉选 `一级护理`，状态下拉选 `在住` → 组合筛选
7. 双击任意老人行 → 弹出信息弹窗（17字段含床位号）
8. 点 **「关闭」** → 回到列表

---

## 第二幕：入住登记 · Nurse 端（~50s）

**操作步骤**

9. 退出 Admin → 输入 nurse 账号登录 → 进入护工端
10. 左侧点 **「老人信息」** → 点击 **「老人入住」**
11. 姓名输入 `王建国`
12. 身份证输入 `120101195006151232` → **自动提取并锁定**出生日期/年龄/性别
13. 填写电话 `13800001234`，选择空闲床位
14. 点 **「入住」** → 登记完成
15. 回到列表，双击王建国行 → 验证信息正确，关闭弹窗

**演示要点**

- 身份证前置在年龄/出生日期之前
- 验证通过后三字段锁定（不可编辑）
- 自动识别性别、计算年龄

---

## 第三幕：外出申请 · Nurse 端（~30s）

**操作步骤**

16. 左侧点 **「外出申请」** → 点击 **「新增外出申请」**
17. 选择老人 `王建国`
18. 选择明天日期 → **预计归日自动设为下一天**
19. 事由填 `家属探访`，点 **「提交」**
20. 列表中显示申请，状态：`待审批`

**演示要点**

- 外出日期变更后归日自动 +1 天

---

## 第四幕：外出审批 · Admin 端（~30s）

**操作步骤**

21. 退出 Nurse → 重新登录 admin
22. 左侧点 **「外出手续」** → 看到王建国申请
23. 双击王建国行 → 弹出老人详情确认身份
24. 点 **「审批通过」** → 弹出二次确认 `确认通过此外出申请？`
25. 点 **「是」** → 审批完成，状态变 `通过`
26. 再次点审批通过 → 提示 `该申请已审批，无法再次审批`

**演示要点**

- 双击行人确认
- 审批前二次确认弹窗
- 已审批记录锁定不可再改

---

## 第五幕：归来 + 退住 · Nurse 端（~30s）

**操作步骤**

27. 切回 Nurse 端（已登录），左侧点 **「外出申请」**
28. 选中王建国 → 点 **「登记归来」** → 实际归时自动填入
29. 左侧点 **「退住申请」** → 点击 **「新增退住申请」**
30. 选择 `王建国`，退住类型选 `正常退住`，填退住日期
31. 点 **「提交」** → 状态：`待审批`

---

## 第六幕：退住审批 · Admin 端收尾（~20s）

**操作步骤**

32. 切回 Admin 端，左侧点 **「退住管理」**
33. 看到王建国退住申请，状态 `待审批`
34. 点 **「审批通过」** → 确认弹窗 → 点 **「是」**
35. 老人状态变 `退住`，床位自动释放为 `空闲`
36. 点击 **「仪表盘」** → 在住老人数 -1，床位使用数变化
